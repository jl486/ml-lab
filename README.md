# ml lab

A simple, beginner-friendly platform designed to make machine learning concepts clear.

## Features

* Well-organized courses covering ML topics from beginner to advanced
* Short, digestible articles unlike textbooks
* Easily-accessible glossary of unfamiliar terms

## Getting started

### Prerequisites

* Install [Node.js](https://nodejs.org/en) which includes [npm](https://docs.npmjs.com/)

### Installation

Clone the repo:

```
git clone https://github.com/jl486/ml-lab.git
```

Install dependencies:

```
npm install
```

Start a local development server:

```
cd ml-lab
npm run dev
```
